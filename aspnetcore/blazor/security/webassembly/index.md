* <span data-ttu-id="5587f-121">Los tokens tienen una duración corta, una hora de forma predeterminada, lo que limita la ventana de ataque.</span><span class="sxs-lookup"><span data-stu-id="5587f-121">Tokens have a short lifetime, one hour by default, which limits the attack window.</span></span> <span data-ttu-id="5587f-122">Los tokens también se pueden revocar en cualquier momento.</span><span class="sxs-lookup"><span data-stu-id="5587f-122">Tokens can also be revoked at any time.</span></span>
* <span data-ttu-id="5587f-123">Los JWT independientes ofrecen garantías al cliente y al servidor sobre el proceso de autenticación.</span><span class="sxs-lookup"><span data-stu-id="5587f-123">Self-contained JWTs offer guarantees to the client and server about the authentication process.</span></span> <span data-ttu-id="5587f-124">Por ejemplo, un cliente tiene los medios para detectar y validar que los tokens que recibe son legítimos y se han emitido como parte de un proceso de autenticación determinado.</span><span class="sxs-lookup"><span data-stu-id="5587f-124">For example, a client has the means to detect and validate that the tokens it receives are legitimate and were emitted as part of a given authentication process.</span></span> <span data-ttu-id="5587f-125">Si un tercero intenta cambiar un token en medio del proceso de autenticación, el cliente puede detectar el token cambiado y evitar usarlo.</span><span class="sxs-lookup"><span data-stu-id="5587f-125">If a third party attempts to switch a token in the middle of the authentication process, the client can detect the switched token and avoid using it.</span></span>
* <span data-ttu-id="5587f-126">Los tokens con OAuth y OIDC no se basan en que el agente de usuario se comporte correctamente para asegurarse de que la aplicación sea segura.</span><span class="sxs-lookup"><span data-stu-id="5587f-126">Tokens with OAuth and OIDC don't rely on the user agent behaving correctly to ensure that the app is secure.</span></span>
* <span data-ttu-id="5587f-127">Los protocolos basados en tokens, como OAuth y OIDC, permiten autenticar y autorizar aplicaciones independientes y hospedadas con el mismo conjunto de características de seguridad.</span><span class="sxs-lookup"><span data-stu-id="5587f-127">Token-based protocols, such as OAuth and OIDC, allow for authenticating and authorizing hosted and standalone apps with the same set of security characteristics.</span></span>
